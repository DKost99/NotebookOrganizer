# Notebook Organizer

Консольное приложение для управления заметками, разработанное в рамках итоговой аттестации. 

## Автор: Даниил Костюхин
## Функционал программы
* Поддержка текстовых заметок (TextNote) с возможностью расширения под другие типы.
* Полный CRUD: создание, чтение, редактирование и удаление записей.
* Стек отмены (UndoStack): отмена последнего изменения благодаря хранению состояний в стеке.
* Фильтрация: поиск заметок по тегу или дате создания.
* Сохранение данных: автоматическая загрузка базы при старте и сохранение в файл data/notes.json.
* Валидация данных: контроль пустых полей и проверка формата даты YYYY-MM-DD.

## Структура проекта
* main.py — Управление логикой приложения и валидацией ввода.
* models/ — Модели данных Note, TextNote, VoiceNote.
* views/ — Компоненты для вывода информации в консоль.
* services/ — Бизнес-логика приложения и работа с JSON.
* tests/ — Модульные тесты для проверки кейсов.
* data/ — Хранилище файлов JSON.

## Инструкция по запуску
1. Перейти в корневую директорию проекта:
cd NotebookOrganizer

2. Запустить приложение:
python main.py
